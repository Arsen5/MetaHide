# MetaHide
C# Разработка стеганографического средства сокрытия в метаданных изображения

инструмент для внедрения и извлечения текста в EXIF метаданных изображений. Разрабатывается для изучения основ стеганографии.

цель проекта
ОРГАНИЧЕНИЯ: Полезная информация не менее 1 Кб. Не используются изображения с авторским правом. ТРЕБОВАНИЯ: - Drag-and-drop интерфейс
- PNG/JPG форматы
- Сохранение без потери качества
- Точное извлечение данных

Ожидаемые результаты
- Работоспособное приложение, реализующее внедрение и извлечение информации в EXIF
- Поддержка различных форматов изображения 
- Инструкция по использованию средства
